## Desafio de Desenvolvimento Web

### Neste trabalho deveríamos de desenvolver um site do zero, que tenha com o objetivo auxiliar o usuário de alguma forma em relação á saúde.

### Com tudo o que houve no Rio Grande do Sul, com a situação das enchentes no mês de Maio, decidimos fazer um site que uniria técnicos dipostos a ajudar pessoas que perderam eletrodomésticos durante as enchentes, fornecendo serviços de manutenção e limpeza de tais eletrodomésticos.

### Este trabalho foi feito em dupla, deixo o link do Github da minha dupla.
